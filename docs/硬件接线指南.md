# 硬件接线指南

> 适用于：智能教室测控系统（Raspberry Pi 5）
>
> 本文档为骨架版本，后续接线细节、实物照片与故障排查请逐步补充。

## 总览

- 主控：Raspberry Pi 5
- 传感器：DHT22（温湿度）、BH1750（光照）
- 执行器：SG90 舵机、红外发射管
- 信号采集：VS1838B 红外接收头
- 接口：5V / 3.3V 供电、GND 共地、I2C-1、GPIO

引脚速查（详见 `config.py`）：

| 模块 | GPIO | 物理 Pin | 说明 |
|------|------|----------|------|
| DHT22 DATA | GPIO17 | Pin 11 | 单总线 |
| BH1750 SDA/SCL | I2C-1 | Pin 3 / Pin 5 | 地址 0x23 |
| 红外接收 OUT | GPIO4 | Pin 7 | lgpio 输入 |
| 红外发射 DAT | GPIO18 | Pin 12 | 38 kHz 载波 |
| 舵机信号 | GPIO23 | Pin 16 | gpiozero 软 PWM |
